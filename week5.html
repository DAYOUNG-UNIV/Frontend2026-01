팀원은 본인 컴퓨터에서 아래 부품 코드를 수정하여 업로드합니다.

🎨 CSS 디자인 필수 가이드

  • 카드 나란히 배치: float: left; 와 margin을 이용해 카드를 가로로 배치했습니다.
  • 부모 높이 붕괴 방지: 팀장 코드의 <main> 태그에 있는 clearfix 클래스가 float으로 인한 에러를 막아줍니다.
  • 힙한 포스트잇 느낌: transform: rotate(-3deg); 로 살짝 삐뚤게 붙인 효과를 주었습니다.

Step 4-1. 예시 코드 CSS 완벽 해부

복사한 뼈대 코드 안에 5주차 핵심 이론이 어떻게 녹아있는지 확인해봅시다!

마법의 배경 고정

body {
  background-attachment: fixed;
}

코르크 보드 이미지가 스크롤을 내려도 화면에 못 박힌 듯 고정됩니다. 스크롤 시 포스트잇만 미끄러지듯 올라가는 입체적인 효과를 만들어냅니다.

찰싹 붙는 제목

header {
  position: sticky;
  top: 20px;
}

제목 박스가 평소엔 정상적으로 배치되어 있다가, 화면 상단 20px 위치에 닿는 순간 fixed처럼 끈적하게 고정되어 따라옵니다.

가로 배치와 방파제

.card { float: left; }

/* 부모 높이 붕괴 방지 */
.clearfix::after {
  clear: both;
}

카드를 float으로 띄워 가로로 나열했습니다. 이로 인해 부모 컨테이너의 높이가 0이 되는 현상을 막기 위해 clearfix 방파제 마법을 부렸습니다.

포스트잇 애니메이션

.card { transform: rotate(-3deg); }
.card:hover {
  transform: scale(1.05) rotate(0deg);
  z-index: 10;
}

기본적으로 삐딱하게(-3도) 붙여둔 카드가, 마우스를 올리면 똑바로(0도) 서면서 약간 커집니다. 이때 z-index를 높여 주변 카드를 가리고 맨 위로 올라오게 만듭니다.

필수 구현 미션 2가지

  • 1. 배지 달기
    position: absolute를 사용하여 내 카드 모서리에 스티커/배지(예: Hot🔥, D-30 등)를 띄워보세요. (부모 요소에 relative 주는 것 잊지 마세요!)
  • 2. 비밀 메모
    카드 맨 아래에 visibility: hidden으로 TMI 메모를 숨겨두고, 마우스를 올렸을 때(:hover) 나타나게 만드세요.

과제 제출 방법

  • 팀 레포지토리 전체를 건드릴 필요 없습니다. 본인이 작성한 카드 1개 분량의 HTML + CSS 코드만 마크다운에 복사하세요.
  • 필수 스크린샷 2장:
    ① 배지가 띄워진 내 카드 캡처
    ② 마우스를 올려 비밀 메모가 나타난 캡처
  • 마크다운 작성 후 PDF로 내보내어(Export) 제출!
